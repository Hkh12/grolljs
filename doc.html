Tutorial

Read everything about groll.js

Intro

groll is a lightweight and powerful library for making scrolling fun. groll contains some jquery plug-ins to apply effect to elements.

Install

First of all, include jquery and lodash into your HTML:

<script src="https://code.jquery.com/jquery-3.2.1.min.js"></script>
<script src="https://cdn.jsdelivr.net/lodash/4.17.4/lodash.min.js"></script>

Download

to install groll, download it from github.

<script src="groll.min.js" type="text/javascript"></script>

CDN

if you want to use groll online, include this code into your html:

<script src="https://rawgit.com/Hkh12/grolljs/master/dist/groll.min.js" type="text/javascript"></script>

Github

.getGrolled()
none

use this method to get the amount of scrolled pixels at the moment.

Example
$(window).getGrolled()

.grollTranslate3d()
number (default: 3)

css translate is a great way to create parallax effect on scroll.
by using this method, use can give better look th your elements.

Example
$("#bigHeader").grollTranslate3d()

This one, has a depth of three, because there is no argument. If you want to change the depth (for example, 2), give an integer to method:

$("#bigHeader").grollTranslate3d(2)

.grollToTop()
number (speed)

This method scrolls an element to top (0 pixel), in a defined speed.

Example
$("html, body").grollToTop()
☝ This one has no speed
we want the page, to scroll to top in 3 seconds:
$("html, body").grollToTop(3000)

.grollLiveEffect()
anonymous function( argument: undefined variable ), number ( speed to update (milliseconds) , default: 1000 )

This method, can do functions, based on the scrolled pixels of page, at the moment.

Example
$("body").grollLiveEffect(function(a){
    console.log(a)
    writes the scrolled pixels at the moment. (updates at every 0.1 seconds)
})

So, this one was a very basic example. now we wanna go higher and make effects with this method.

Imagine: you want to style your fixed navigation bar. you want to make a

background-color: rgba(255,0,0,(a number based on scrolled pixels))

for the navbar.
Groll is here to rescue!

$("#myNav").grollLiveEffect(function(foo){
    if (foo < 250){
        var a = foo / 250;
        $("#myNav").css('background-color', 'rgba(255,0,0,' + a + ')')
    }
}, 0.1)

.grollEvent()
Object (offset* , class to be added, class to be removed)

this method is used to modify the classes of an element, after scrolling more than a defined offset.

Example
$('#navbar').grollEvent({
    offset: 200,
    addClass: 'colored',
    removeClass: 'transparent'
})

Fact: the example above, is a great effect for your fixed navigation bars, that you can use it for landing pages.

.grollTo()
number (target*, speed)

this method is used to scroll an element, to defined target (pixels), with a defined speed.

Example
$("html, body").grollTo(100)

This one, scrolls the page, without any speed.
but, we want it to scroll to defined target in 2 seconds, so we have:

$("html, body").grollTo(100, 2000)

.grollToBottom()

this method, follows the structures of .grollToTop() method, but with one difference. according to their names, the previous method scrolls the element to top; but this one scrolls to the bottom.
