Groll.js v1.0alpha

Lightweight jQuery Plug-ins
to make scrolling more fun.
 
Today, web users will never waste their time. They only look at pages that are interesting to them. As a UI designer, you need to get the user attracted to you.
Scrolling may be a normal thing; but by maneuvering it, you can create a beautiful user interface.
Parallax effects, sudden changes when scrolling down, all of this can help you design.
Groll.js is a lightweight and powerful library for making scrolling more fun for the users, And can do everything that you want.
